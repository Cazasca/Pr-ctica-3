Etiquetas HTML

  • <dl> <\dl> Usadas para crear una lista de definiciones.
  • <dt> <\dt> Representa un término definido por la siguiente etiqueta <dd>
  • <dd> <\dd> Se usa para definir los términos listados antes que él.

    Por ejemplo:
    <dl>

    <dt>HTML<\dt>
    <dd>Siglas en inglés de HyperText Markup Language.<\dt>
    <dd>Significa lenguajes de marcas de hipertexto.<\dd>
    <\dl>

  • <blockquote> </blockquote> Crea citas en bloque, marca las citas a otros autores o documentos.

    Por ejemplo:
    El lenguaje HTML es <blockquote>

    el componente más básico de la Web. Define el significado y la estructura del contenido web.
    <\blockquote>

  • <q> <\q> Indica que el texto adjunto es una cita corta en línea.

    Por ejemplo:
    La mayoría de páginas web tienen <q>por ejemplo, una página de inicio, otra de producto, otra de contacto, etc.<\q>

  • <cite> <\cite> Marca una referencia a una fuente, o el autor de un texto citado.

    Por ejemplo:
    <cite>Galileo<\cite> dijo: "... y sin embargo, se mueve."

  • <abbr> <\abbr> Representa una abreviación o acrónimo.

    Por ejemplo:
    Esto fue crado por <abbr>HTML<\abbr>

  • <address> <\taddress>Aporta información de contacto para <article> más cercano o ancestro <body>.

    Por ejemplo:

    You can contact author at www.somedomain.com.
    If you see any bugs, please contact webmaster.
    You may also want to visit us:
    Mozilla Foundation
    1981 Landings Drive
    Building K
    Mountain View, CA 94043-0801
    USA

  • <sub> <\sub> Utilizado para los subíndices.

    Por ejemplo:
    La fórmula química del oxono es: O<sub>3<\sub> <\p>

  • <sup> <\sup> Utilizado para los superíndices.

    Por ejemplo:
    El C<sup>14<\sup> se utiliza para la determinación de la edad de un vestigio orgánico.

  • <code> <\code> Apropiado para marcar el código de un programa.

    Por ejemplo:
    <code>
    #include main() { for(;;) { printf ("Hello World!\n"); } }
    <\code>

  • <pre> <\pre> Representa texto preformateado.

    Por ejemplo:
    <pre> 

                    body{
                  color:  red;
                }
                a   {
                  color:green;
                }
    <\pre>

  • <var> <var>Su función es resaltar texto en el documento web, indicando que el mismo son la definición de variables en un programa de ordenador.

    Por ejemplo:
    El programa que se está analizando es el siguiente
    <var>int x;<var>
    <var>int y;<var>
    Se ha solicitado la creación de dos variables


  • <kdb> <kdb> Permite especificar un formato de texto para distinguir lo que el usuario tendría que teclear en la solicitud de un programa.

    Por ejemplo:
    Ha de teclear una dirección valida de internet:
    www.aprendoencasa.com a continuación pulsar Intro

  • <samp> <samp> Se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador.

    Por ejemplo:
    <samp>Keyboard not found press F1 to continue<samp>

  • <time> <\time> Representa un periodo específico en el tiempo.

    Por ejemplo:
    La pelicula empieza a las <time>20:00<\time>
